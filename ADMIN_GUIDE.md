# 初衷小鹿 — 管理後台操作說明

## 如何進入後台

1. 先登入您的 Manus 帳號（點擊導覽列右上角登入按鈕）
2. 登入後，在瀏覽器網址列輸入：`/admin`
3. 系統會自動驗證您是否為管理員（Owner 帳號自動擁有 admin 權限）

---

## 後台功能總覽

| 頁面 | 路徑 | 功能 |
|------|------|------|
| 儀表板 | `/admin` | 查看訂單數、營收、會員數、商品數統計 |
| 商品管理 | `/admin/products` | 新增、編輯、上下架商品 |
| 訂單管理 | `/admin/orders` | 查看訂單詳情、更新訂單狀態 |
| 會員資料 | `/admin/members` | 查看會員名單、管理角色 |

---

## 商品管理

### 新增商品

1. 前往 `/admin/products`
2. 點擊右上角「新增商品」按鈕
3. 填寫以下欄位：
   - **商品名稱**（必填）
   - **分類**：選擇推薦組合 / 居家鍋底 / 牛 USDA Prime Beef / 豬 Black Pig / 雞 Chicken / 羊 Lamb / 海鮮 Seafood
   - **售價**（必填，格式如：`1200`）
   - **原價**（選填，用於顯示折扣）
   - **庫存數量**
   - **商品描述**
   - **圖片網址**（可留空，系統會依商品名稱自動匹配圖片）
   - **標籤**（如：熟成、頂級、季節限定）
   - **精選商品**：勾選後會顯示「精選」徽章
   - **上架狀態**：勾選後商品才會在商店顯示
4. 點擊「儲存」

### 編輯商品

- 在商品列表中點擊「編輯」按鈕
- 修改後點擊「儲存」

### 上下架商品

- 在商品列表中，點擊「上架」/「下架」按鈕即可切換狀態
- 下架商品不會顯示在商店頁面

## 訂單管理

### 查看訂單

1. 前往 `/admin/orders`
2. 可看到所有訂單，包含：
   - 訂單編號
   - 客戶姓名 / 電話
   - 訂購商品與數量
   - 金額
   - 付款狀態（待付款 / 已付款 / 已取消）
   - 訂單狀態（待確認 / 處理中 / 已出貨 / 已完成）

### 更新訂單狀態

1. 找到要更新的訂單
2. 點擊狀態下拉選單
3. 選擇新狀態：
   - `pending` → 待確認
   - `processing` → 處理中（已確認，準備出貨）
   - `shipped` → 已出貨
   - `completed` → 已完成
   - `cancelled` → 已取消

## 會員管理

### 查看會員

1. 前往 `/admin/members`
2. 可看到所有已登入過的會員資料

### 設定管理員

若需要讓其他人也能管理後台：

1. 前往右側管理介面 → **Database（資料庫）**
2. 找到 `users` 資料表
3. 找到目標使用者的記錄
4. 將 `role` 欄位從 `user` 改為 `admin`

## Stripe 金流

- 測試付款請使用信用卡號：`4242 4242 4242 4242`，任意有效期（如 12/28）和 CVV（如 123）
- 付款成功後，訂單狀態會自動更新為「已付款」

## 常見問題

**Q：為什麼我看不到後台？**
A：請確認您已登入，且您的帳號 role 為 `admin`。Owner 帳號（第一個登入的帳號）會自動設為 admin。

**Q：商品圖片如何更新？**
A：在商品編輯頁面的「圖片網址」欄位填入圖片的 URL，或留空讓系統依商品名稱自動匹配預設圖片。

**Q：訂單通知如何設定？**
A：每筆新訂單成立時，系統會自動發送通知至 Manus 平台的通知中心。
